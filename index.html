JavaScriptで実現！動的なローディング画面のデザインアイデア

ユーザーを飽きさせない、DOM操作によるインタラクティブなローディングアニメーションの作り方をご紹介します。

はじめに：なぜローディング画面が重要なのか？

Webサイトやアプリケーションの読み込み中、ユーザーは何も操作できない状態になります。この間にただ真っ白な画面を表示するだけでは、ユーザーは離脱してしまう可能性があります。そこで重要になるのが、ローディング画面です。

ローディング画面は、読み込み中の待ち時間を視覚的に埋めるだけでなく、ユーザーに「コンテンツが準備されている」という安心感を与え、期待感を高める役割も果たします。

今回使用するJavaScriptとDOM操作の基本知識

  • 新しいHTML要素の作成

    JavaScript

    document.createElement('div');
    これは、新しいHTML要素（この場合は`div`要素）をJavaScript内で作成するためのメソッドです。作成された要素はまだドキュメントには追加されていません。
  • CSSクラスの追加

    JavaScript

    element.classList.add('className');
    これは、HTML要素にCSSクラスを追加するためのメソッドです。これにより、CSSで定義されたスタイルをその要素に適用できます。
  • インラインスタイルの直接設定

    JavaScript

    element.style.propertyName = 'value';
    これは、HTML要素のインラインスタイルを直接設定するための方法です。CSSプロパティをJavaScriptから直接操作できます。例えば、`backgroundColor`、`position`、`width`、`height`、`zIndex`などがあります。
  • 要素を先頭に追加

    JavaScript

    document.body.prepend(element);
    これは、指定された要素（`element`）を``要素の最初の子要素として追加するためのメソッドです。
  • 要素を末尾に追加

    JavaScript

    document.body.append(element);
    これは、指定された要素（`element`）を``要素の最後の子要素として追加するためのメソッドです。
  • 要素を子要素として追加

    JavaScript

    document.body.appendChild(element);
    これもまた、指定された要素（`element`）を``要素の子要素の最後に追加するためのメソッドです。`append`と似ていますが、`appendChild`はNodeオブジェクトのみを受け取り、一度に1つのノードしか追加できません。
  • 繰り返し処理の設定

    JavaScript

    setInterval(function, delay);
    これは、指定された関数を一定の間隔（`delay`ミリ秒ごと）で繰り返し実行するための関数です。戻り値としてインターバルIDを返します。
  • コンソールへの出力

    JavaScript

    console.log("message");
    これは、開発者ツールのコンソールにメッセージを出力するための関数です。デバッグ目的で、プログラムの実行フローや変数の値を確認するのに役立ちます。

DOM操作：①画面全体にオーバーレイを表示しよう！

まず初めに何らかの処理が開始された際に、ユーザーに『今、動いていますよ！』と明確に伝えるために、画面全体を一時的に白くするテクニックを実装します。 このシンプルな表示によって、ユーザーは安心して処理の完了を待つことができるようになります。
また、この白い画面が表示されている間は、ユーザーが誤って他の要素をクリックしても影響がないように設定する方法も合わせて見ていきます。 とても実用的なテクニックなので、ぜひマスターしてください。

以下のコードは、白いオーバーレイを画面全体に表示するためのJavaScriptです。

overlay.js

// 白いスクリーン用のdivを作成
const whiteOverlay = document.createElement('div');

// whiteOverlayにクラスを追加
whiteOverlay.classList.add('white-overlay');

// スタイルを直接適用
whiteOverlay.style.position = 'fixed'; // 固定位置にする
whiteOverlay.style.top = '0'; // 画面の上部に配置
whiteOverlay.style.left = '0'; // 画面の左端に配置
whiteOverlay.style.width = '100%'; // 画面全体の幅
whiteOverlay.style.height = '100%'; // 画面全体の高さ
whiteOverlay.style.backgroundColor = 'rgba(255, 255, 255, 0.5)'; // 半透明の白
whiteOverlay.style.zIndex = '999'; // コンテンツより上に表示する
whiteOverlay.style.pointerEvents = 'auto'; // クリックを無効にする（デフォルトはautoですが、明示的に指定）

document.body.appendChild(whiteOverlay); // bodyに追加

まず、const whiteOverlay = document.createElement('div'); の行では、 HTMLの div 要素が新しく作成され、それが whiteOverlay という名前の定数に格納されます。 この新たに作成された div 要素が、ユーザーが操作している画面全体を覆う白いスクリーンとして機能します。

次に、whiteOverlay.classList.add('white-overlay'); という記述で、 この div 要素に white-overlay というCSSクラスが追加されます。 このクラスは、別途定義されたCSSスタイルを適用するために使用され、 またJavaScriptコード内で特定のオーバーレイ要素を識別する際にも役立ちます。

その後、複数の行で whiteOverlay.style プロパティを使用して、 このオーバーレイの視覚的なスタイルが直接適用されています。

ポイント

ここでのポイントは、JavaScriptによるDOM操作で、画面全体を覆う半透明のオーバーレイを動的に生成し、 視覚的なスタイルを適用してHTMLに組み込んでいる点です。 特に、position: fixedとz-indexを用いて、他のコンテンツの上に固定表示させる仕組みが重要です。

DOM操作：②レターボックスを追加しよう！

映画のような演出をWebに取り入れてみましょう！ レターボックスとは、画面の上下に黒い帯を表示することで、映像にシネマティックな印象を与える手法です。 これをローディング画面に応用することで、ユーザーに「特別な体験が始まる」という期待感を演出できます。
このセクションでは、JavaScriptを使って画面の上下に黒い帯（レターボックス）を動的に追加する方法を紹介します。 DOM操作によって、読み込み中の画面に映画のような雰囲気を加えることができます。

letterbox.js

// レターボックス上部用のdivを作成
const letterboxTop = document.createElement('div');
letterboxTop.classList.add('letterbox-top');
// 黒い四角を表示するためのスタイルを直接適用
letterboxTop.style.backgroundColor = 'rgba(0, 0, 0, 1)';
letterboxTop.style.position = 'fixed';
letterboxTop.style.left = '0';
letterboxTop.style.width = '100%';
letterboxTop.style.height = '10vh'; // 画面の高さの"10"%
letterboxTop.style.top = '0';
letterboxTop.style.zIndex = '1000'; // 他の要素より手前に表示(オーバーレイより上)

// レターボックス下部用のdivを作成
const letterboxBottom = document.createElement('div');
letterboxBottom.classList.add('letterbox-bottom');
// 黒い四角を表示するためのスタイルを直接適用
letterboxBottom.style.backgroundColor = 'rgba(0, 0, 0, 1)';
letterboxBottom.style.position = 'fixed';
letterboxBottom.style.left = '0';
letterboxBottom.style.width = '100%';
letterboxBottom.style.height = '15vh'; // 画面の高さの"15"%
letterboxBottom.style.bottom = '0';
letterboxBottom.style.zIndex = '1000'; // 他の要素より手前に表示(オーバーレイより上)

// レターボックスの要素をbodyに追加
document.body.prepend(letterboxTop);
document.body.append(letterboxBottom);

まず、const letterboxTop = document.createElement('div'); の行では、 HTMLの div 要素が新しく作成され、 それが letterboxTop という名前の定数に格納されます。 この要素は、画面の上部に黒い帯を表示するためのレターボックスとして機能します。

次に、letterboxTop.classList.add('letterbox-top'); によって、 この div 要素にletterbox-top というCSSクラスが追加されます。 このクラスは、スタイルの適用や要素の識別に使用されます。

その後、letterboxTop.style を使って、背景色や位置、サイズなどのスタイルを直接設定しています。 特に position: fixed によって、スクロールしても常に画面上部に表示されるようになっています。

同様に、const letterboxBottom = document.createElement('div'); によって、 下部用のレターボックス要素が作成され、letterbox-bottom クラスが追加されます。 bottom: 0 によって画面の下端に固定表示され、height: 15vh によって高さが画面の15%に設定されています。

最後に、document.body.prepend(letterboxTop); と document.body.append(letterboxBottom); によって、 それぞれのレターボックス要素がHTMLの body に追加され、画面の上下に黒い帯が表示されるようになります。

ポイント

ここでのポイントは、JavaScriptによるDOM操作で、画面の上下に黒い帯（レターボックス）を動的に生成し、 映画のような演出をWeb上で再現している点です。 特に、position: fixedとz-indexを用いて、 他のコンテンツの上に固定表示させる仕組みが重要です。 また、vh単位を使うことで、画面サイズに応じた柔軟な高さ設定が可能になっています。
